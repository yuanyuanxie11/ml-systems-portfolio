# Course assignment briefs

The two PDFs in this folder are the original Northwestern MLDS 423
assignment briefs that scoped this project. They are kept in the repo for
provenance — they explain why the course-version of this work
(`README_academic.md` at the repo root) is structured the way it is.

These briefs are **not** required reading to use the repo. The portfolio
narrative lives in:

- `../../README.md` — production framing
- `../../README_academic.md` — original course deliverable
- `../../ARCHITECTURE.md` — AWS production design
- `../../MODEL_CARD.md` — model card
- `../API.md` — REST API spec
- `../MLOPS.md` — MLOps pipeline

Files here:

- `Netflix Prize Data Mining Project.pdf` — main brief (English)
- `Netflix Prize Data Mining Project – 5 人分工计划.pdf` — five-person workplan (Chinese)
